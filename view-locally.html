View TyleDeClouds Website Locally

To view the TyleDeClouds website locally, you have two options:

  1. Simple Option: Double-click the index.html file and open it in your browser.
    Note: Some browsers may restrict loading local images due to security constraints.
  2. Better Option: Use a local server to serve the website:
    • If you have Python installed, run: python -m http.server in this directory
    • If you have Node.js installed, run: npx serve
    • If you have PHP installed, run: php -S localhost:8000
    Then visit http://localhost:8000 in your browser
